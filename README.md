# Important Commands
- `TSInstall <arg>` install tree-sitter for arg
- `Mason` open mason menu

# Important shortcuts
| Keybind | Description |
|-|-|
| `<leader>e` | Explore Files |
| `<leader>ff` | Find File |
| `<leader>fw` | Find Word |

# Font
